The 'a' tag is really important here. It builds links from this page to another page, which could be inside this folder/directory/site/server, or to another site sits somewhere on the Internet. Be sure to include either 'http://' or 'https://' in that case. Or the browser will try to find it from the folder of current page, which is obviously wouldn't find anything here, and therefore return an error message. On the other hand, if you want to link index page to about page and all the other pages inside your site, 'a' tag is the way to put them together, so that you got hyperlink and get them working together to form a site.

Another thing here is inline style used, be it 'em' or 'strong' or 'a' tag.
